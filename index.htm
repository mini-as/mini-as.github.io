Hello and Welcome to official homepage of minis operating system.
The operating system is not ready yet but Minis assembler can be
used for compiling Minis OS and cpu instructions of intel (80386)
or better and can generate 16 or 32 code in just 2 passes.
This assembler support almost all instruction of intel (80386)
and create only flat binary file.
The source code of this assembler written in assembly langauge, the
only way I know how to do that, because for his fast, simple
and small size.
