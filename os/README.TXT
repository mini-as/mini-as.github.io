What is minis:
minis is small operating system, designed to work on 
computer with processor intel (80386) or better.
written by me as hobby for fun and learning how
computers work.
minis assembler also written by me, and i make it as simple
as possible and small.
The name minis mean mini system.

To build minis.img you need:
minis assembler version 0.09
install imdisk and click on build.bat

To test on Bochs x86 PC emulator:
download and install Bochs-2.7
click on bochsrc.bxrc
if you see desktop and cmd icon,
you have successfully booted the system :-)
